Pipeline Metodológico de Preparação

1

Tratamento de Zeros

Limpeza dos Dados

2

Divisão Estratificada

Treino, Validação, Teste

60% Treino • 20% Validação • 20% Teste
Mantém proporção das classes em cada conjunto
3

Remoção de Outliers

Método IQR

IQR = Q3 - Q1
Outliers: < Q1 - 1.5×IQR ou > Q3 + 1.5×IQR
Remove valores extremos que podem distorcer o modelo
4

Balanceamento de Classes

Técnica SMOTE

65% → 50% cada classe
Synthetic Minority Oversampling TEchnique - gera exemplos sintéticos da classe minoritária
5

Normalização dos Dados

Z-Score

Z = (X - μ) / σ
Padroniza features para média = 0 e desvio = 1
Dados → Limpeza → Divisão → Outliers → Balanceamento → Normalização → Modelo

Como funciona o SMOTE:

1. Seleciona ponto da classe minoritária
2. Encontra K vizinhos mais próximos
3. Cria ponto sintético entre eles
4. Repete até balancear as classes

Pré-processamento de Dados

Principais Achados

I

Distorção por imputação simples

A imputação pela mediana foi descartada por distorcer a distribuição dos dados, criando picos artificiais que reduzem a variância e podem ocultar padrões relevantes.

II

Ineficiência de métodos complexos

As imputações por KNN e Regressão Linear, apesar de sofisticadas, não ofereceram melhorias que justificassem sua complexidade e o risco de gerar dados sintéticos.

III

Perda de poder estatístico por exclusão

A remoção de colunas e linhas com dados zero causou uma perda crítica de informação e poder estatístico, enfraquecendo os modelos.

IV

Eficácia da abordagem robusta

A combinação de dados brutos com um modelo robusto (Random Forest) foi a estratégia superior, lidando eficazmente com o ruído sem manipulação artificial dos dados.

Conclusões

A validação de comportamento clínico é indispensável e supera a avaliação baseada apenas em métricas.

Abordagens complexas de pré-processamento podem não garantir um resultado superior.

O modelo Random Forest foi a solução mais performática e segura para este estudo.
